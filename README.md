# RPG Simulator

Консольний RPG-симулятор - навчальний проект з об'єктно-орієнтованого програмування.

---

## Опис проекту

Консольна гра, де гравець бореться з випадковими ворогами (гоблін, орк, троль), отримує досвід, підвищує рівень та покращує характеристики.

---

## Архітектура

RpgSimulator/
├── RpgSimulator.Console/     Точка входу (UI)
├── RpgSimulator.Domain/      Сутності, інтерфейси, патерни
├── RpgSimulator.App/         Логіка та сервіси
└── RpgSimulator.Tests/       Модульні тести

---

## Використані патерни проектування

| Патерн | Призначення |
|--------|-------------|
| Factory Method | Створення типів ворогів (Enemy.Factory) |
| Strategy | Різні стратегії атаки (Normal, Critical) |
| Observer | Сповіщення про події в бою |
| Repository | Універсальне сховище (IRepository<T>) |

---

## SOLID принципи

| Принцип | Реалізація |
|---------|-------------|
| SRP | Розділення Entity, CombatService, Logger |
| OCP | Додавання нових стратегій без зміни існуючого коду |
| LSP | Класи Player та Enemy наслідують Entity |
| ISP | Інтерфейси IAttackable, IObserver, IObservable |
| DIP | Залежність від IAttackStrategy, а не конкретних класів |

---

## Ключові можливості

- Бій з випадковим ворогом
- Система досвіду та рівнів
- Стратегії атаки (звичайна, критична)
- Логування подій
- Валідація вхідних даних

---

## Технології

- C# / .NET Framework 4.7.2
- xUnit + Moq (тестування)
